PackHub

Install Linux packages straight from GitHub!

packhub creates virtual linux package repository (apt, dnf, yum etc) on the fly directly from GitHub Releases.

Paste your GitHub repository URL below to get the command to setup packhub repository.

Ubuntu 
wget -qO- http://packhub.dev/sh/ubuntu/github/mominul/pack-exp2 | sh
Debian 
wget -qO- http://packhub.dev/sh/ubuntu/github/mominul/pack-exp2 | sh
