PackHubbeta

Install Linux packages straight from GitHub! 🚀

PackHub creates virtual Linux package repositories (apt, dnf, yum etc) on the fly directly from GitHub Releases.

Developer Freedom

No need to maintain separate repositories or rely on a maintainer—PackHub handles distribution for you.

User Empowerment

Get the apps you need instantly—no waiting for repositories or manual installations.

Seamless Updates

PackHub monitors GitHub releases and automatically delivers updates to your package manager.

Smart Versioning

Detects your system’s version and selects the most compatible package from GitHub Releases.

Paste your GitHub repository URL below to get the command to setup PackHub repository in your system.
